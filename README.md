# Bayesian Non-linear Poisson Regression for Proportion AMR

Example code accompanying Vass et al. (2026), demonstrating a Bayesian 
non-linear Poisson model for estimating proportion antimicrobial 
resistance (AMR) in *E. coli*.

## Contents

- `Example_Poisson_model_Vass2026.Rproj` — RStudio project file
- `Model_Example_code.Rmd` — model fitting and plotting example
- `dummy_AMR_data.rds` — example dataset
- `dummy_model.rds` — pre-fitted example brms model object

- R 
- [RStudio]
